### 🏛️ GovFlow

**[GovFlow](https://github.com/renanbambam/govflow)**
`ASP.NET Core · .NET 8 · PostgreSQL`

Enterprise workflow and process management platform designed around Clean Architecture, DDD and CQRS principles.

**Engineering Highlights**

- Clean Architecture + DDD + CQRS + MediatR
- JWT Authentication + RBAC authorization
- Process timeline and audit history
- Document management with PDF uploads
- SignalR realtime process updates
- Hangfire SLA monitoring and background jobs
- 60 automated tests across Domain, Application and Integration layers
- Dockerized PostgreSQL environment
